# Document provider-specific MCP setup

The README will document generic stdio launch through `heracles mcp serve --config /path/to/heracles.yaml` and provide copy-ready MCP client configurations for Codex, Claude Code, OpenCode, Kimi Code, OpenClaw, and Hermes wherever each client supports MCP. It will explain that Heracles-launched provider sessions receive MCP automatically, list the available high-level tools and the absence of arbitrary shell execution, and include `heracles doctor` plus a direct MCP smoke test for troubleshooting.
