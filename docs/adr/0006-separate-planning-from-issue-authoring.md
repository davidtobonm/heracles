# Separate planning from issue authoring

The Planning Stage will use a configurable Planner to explore target repositories, clarify the problem, lazily maintain necessary documentation, and produce a PRD before an Approval Gate. After approval, a separate Issue Author will propose Heracles-compatible tracer-bullet issues and stop at another Approval Gate before publishing them. Planning uses a default Question Budget of 20; the Planner may finish earlier and must ask permission before exceeding it.
